# 🚀 Portfolio Website — Project Overview

## Tujuan
Membuat website portfolio pribadi yang profesional dan dapat ditemukan di Google, lengkap dengan blog untuk personal branding dan SEO.

## Target Pengguna
- Recruiter / HRD yang mencari developer
- Sesama mahasiswa / developer yang menemukan artikel blog
- Dosen atau pihak akademik yang ingin melihat portofolio
