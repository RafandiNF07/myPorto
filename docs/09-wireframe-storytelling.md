# UX Wireframe & Storytelling Flow
## Rafa's Backend Portfolio

### Core Philosophy
*   **Not a Template:** This is a backend developer's portfolio. No generic typing laptops.
*   **Storytelling Flow:** 
    1. Who is Rafa? (3s) 
    2. What can he build? (10s) 
    3. How does he think? (Blog/Skills) (20s) 
    4. Proof of expertise (Certs/GitHub) (30s)
*   **Abstract Floating Bento:** Elements aren't trapped in a rigid grid. They float, overlap, and draw the eye down the page naturally.

---

### Section 1: The Anchor (Hero - Living System)
**Visual:** 70% of viewport height.
**Background:** Animasi **Living System V2** — aliran paket data (SVG animasi) yang mensimulasikan trafik jaringan sistem ke komponen (Cache, Database, Worker, API, Client).
**Centerpiece (Animated):** Teks nama "Rafandi Nova Fitra" dengan efek *decode matrix*, disusul dengan subtitle "BACKEND ARCHITECTURE", "CLOUD INFRASTRUCTURE", foto profil *monochromatic*, dan *tags*.
**Action:** Transisi *staggered entrance* yang memunculkan Bento Grid di bawahnya.

---

### Section 2: The "Wow" Project (Breaking the Grid)
**Visual:** A massive horizontal card that slightly overlaps the bottom of the Hero section.
**Content:** Supabase Auth Gateway.
**Layout:** 
- Left: A mocked-up terminal window or code snippet block showing the actual Go/Redis code.
- Right: Large Typography title, the impact (e.g., "Reduced latency by 40%"), and a glowing "Read Case Study" button.

---

### Section 3: The Mind (Blog & Philosophy)
**Visual:** A staggered, asymmetrical layout.
**Left Side (Large):** Latest deep-dive article (e.g., "Why I switched to CachyOS"). It takes up space because writing is a priority.
**Right Side (Stack of smaller floating cards):**
- Card 1: 3 recent post titles (Minimalist list).
- Card 2: "Current Focus: Cybersecurity & Network Defense" (With a glowing radar or lock icon).

---

### Section 4: The Arsenal (Skills & Certifications)
**Visual:** A horizontal masonry layout. No boring lists.
**Top Row:** Floating badges for core stack (Python, Go, PostgreSQL, Linux).
**Bottom Row (Certifications):** Dedicated space for AWS, Google, TryHackMe, etc. Designed like physical metallic or glass badges that shine on hover.

---

### Section 5: The Proof & Exit (GitHub & Contact)
**Visual:** Anchoring the bottom of the page.
**Left:** A live GitHub contribution graph (or a stylized version of it) showing activity.
**Right (Call to Action):** "Let's build something." Big, bold email link.

---

### Navigation
*   A glassmorphic pill-shaped navigation bar that floats at the very bottom center of the screen (Mobile style) or top right, appearing only after scrolling past the hero.
